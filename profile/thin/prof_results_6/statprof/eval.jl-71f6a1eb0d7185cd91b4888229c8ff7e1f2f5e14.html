137 """
138 chebjacobian(c::ChebPoly, x)
139
140 Return a tuple `(v, J)` where `v` is the value `c(x)` of the Chebyshev
141 polynomial `c` at `x`, and `J` is the Jacobian of this value with respect to `x`.
142
143 That is, if `v` is a vector, then `J` is a matrix of `length(v)` × `length(x)`
144 giving the derivatives of each component. If `v` is a scalar, then `J`
145 is a 1-row matrix; in this case you may wish to call `chebgradient` instead.
146 """
147 function chebjacobian(c::ChebPoly{N}, x::SVector{N,<:Real}) where {N}
148 x0 = @. (x - c.lb) * 2 / (c.ub - c.lb) - 1
149 all(abs.(x0) .≤ 1) || throw(ArgumentError("$x not in domain $(c.lb) to $(c.ub)"))
150 v, J = Jevaluate(x0, c.coefs, Val{N}(), 1, length(c.coefs))
151 return v, J .* 2 ./ (c.ub .- c.lb)'
152 end
